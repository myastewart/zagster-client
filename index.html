Zagster Bike Share in Bend

About

The goal of this project was to use our understanding of VS code and HTML, css, and JavaScript to create a webpage with a data visualization using data collected from the Zagster Bike Share service. This works by connecting our webpage made with VS code to the data server via a link over the internent. The data visualization requests data from this server and then translates what it recieves into the template that was designed using Chart.js.
